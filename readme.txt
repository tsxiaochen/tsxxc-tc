== Description ==

这是小陈兴趣站所开发的一款首页弹窗公告插件，插件十分简单，可以在首页弹出一个公告框。

= CREDITS =

首页弹窗公告插件最初由tsxcc创建,它现在由tsxcc拥有和维护。

= Usage =

1.下载`homepage-popup-notice.zip`解压到`wp-content/plugins/`
2. 通过 WordPress 中的“插件”菜单激活插件。
3.“仪表板”->“弹窗公告设置”
4.关于这个插件的更多信息，请访问：【插件主页】（https://tsxiaochen.com/wordpress/plugins/1478.html）。
